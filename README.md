# Persian Rug Auctions — Live Demo

A modern auction platform demo for Persian Rug Auctions, showcasing a luxury bidding experience for high-value handmade carpets.

## Live Demo

**View the live site:** https://247af91f.persian-rug-auctions-demo.pages.dev

## Pages

| Page | URL | Description |
|------|-----|-------------|
| **Homepage** | [/](https://247af91f.persian-rug-auctions-demo.pages.dev) | Hero, featured lots, how it works, about section |
| **Catalogue** | [/catalogue](https://247af91f.persian-rug-auctions-demo.pages.dev/catalogue) | Full auction catalogue with filtering |

## Tech Stack

- **Framework:** Next.js 16 + React 19
- **Styling:** Tailwind CSS
- **Typography:** Cormorant Garamond + DM Sans
- **Icons:** Lucide React
- **Animation:** Framer Motion

## Design System

- **Primary:** Navy (#1a2332)
- **Accent:** Gold (#c9a962)
- **Background:** Ivory (#f8f6f2)

## Development

```bash
npm install
npm run dev
```

## Deployment

This site is auto-deployed to Cloudflare Pages on every push to `main`.
